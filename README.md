# vue2-template

## create project
```
vue create hello-world
See [vue-create](https://cli.vuejs.org/zh/guide/creating-a-project.html#vue-create).
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
or
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
